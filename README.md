# yuvplayer 2.0

Lightweight YUV player which supports various YUV format

Supporting Format
* Planar format
* 8bit YUV420 (YV12)
* 8bit YUV422 (YV16)
* 10bit YUV422
* 8bit YUV444
* RGB16
* RGB24
* RGB32


* Interleaved format
* 8bit NV12
* 8bit NV21
* 8bit UYVY
* 10bit UYVY(V210)
* 8bit VYUY

Supporting Feature
* Zoom (4:1 ~ 1:4)
* Store frame to YUV/BMP

Hot key
* "→": frm-forward
* "←": frm-backward
* "↑": zoom-in
* "↓": zoom-out
* "p/space": play/pause toggle
* "o": open
* "g": goto a frame
* "x": exit
* "h" set size to 1080p(HD)
* "s" set size to 720p(SD)
* "c" set size to CIF
* "q" set size to QCIF
